# Генератор пароля на Python
## Описание 
 Данный проект это генератор паролей , разработанный на языке Python в котором используется  библиотеки tkinter для графического интерфейса. В программе реализован две строки в первой пользователь вписывает сколько паролей будет генернироваться за раз , а во второй вписывает сколько символов будет в пароле .

## Возможности
* Графический интерфейс сделаный на tkinter
* возможность самому вприсывать количество символов в пароле и количество самих паролей 
* Поддержка всех основных функций:
  * Строка ввода количества символов в пароле 
  * Строка ввода количества пароле за раз 
  * Выбрать наличие букв в пароле 
  * Выбрать наличие цифр в пароле 
  * Выбрать наличие спец.символов в пароле 
* Кнопка копирования пароля

## Используемые технологии
* Python
* tkinter

 Изучить основы объектно-ориентированного программирования, работы с графическим интерфейсом , методами генерации и создание простых генераторов на Python.
